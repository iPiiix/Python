# Python – Ejercicios y Prácticas de Clase

Este repositorio contiene una colección de ejercicios realizados durante el aprendizaje de Python. Su propósito es documentar el progreso, reforzar conceptos fundamentales y disponer de un espacio centralizado para el estudio, consulta y mejora continua del código.

---

## Objetivos del repositorio

- Practicar los fundamentos del lenguaje Python.
- Comprender y aplicar estructuras de control, funciones y tipos de datos.
- Desarrollar una base sólida para proyectos más avanzados.
- Mantener buenas prácticas de organización y documentación del código.

---
 
## Requisitos

- Python 3.x instalado.
- Editor de texto o IDE recomendado: VSCode, PyCharm o similar.
- Conocimientos básicos de programación (no imprescindibles, pero útiles).
